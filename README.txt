INSTRUCCIONES PARA COMPILAR Y EJECUTAR LA APLICACIÓN
====================================================
1.- AGREGAR UNA VARIABLE DE ENTORNO DEL SISTEMA DE NOMBRE "PruebaTecnica" CON EL VALOR "server=127.0.0.1;uid=root;pwd=;database=pruebatecnica"
2.- EL VALOR DE LA VARIABLE DE ENTORNO DEL SISTEMA "PruebaTecnica" PUEDE VARIAS, INGRESAR LA CADENA DE CONEXION PARA MYSQL CORRESPONDIENTE AL AMBIENTE DE PRUEBAS
3.- PARA EJECUTAR LA APLICACION ES NECESARIO CONTAR CON ACCESO A INTERNET DEBIDO A QUE LA SOLUCION UTILIZA LIBRERIAS EXTERNAS QUE NECESITAN SE DESCARGADAS PREVIO A COMPILAR
4.- LA BASE DE DATOS DEBE ESTAR VACIA NO ES NECESARIO EJECUTAR NINGUN SCRIPT DE BASE DE DATOS, UNICAMENTE EJECUTAR LA SOLUCION Y PROBARLA
